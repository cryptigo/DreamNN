## Installation ##
1. Install python
2. clone the repo
   ```sh
   sudo git clone https://github.com/cryptigo/DreamNN.git
   ```
3. Install requirements
   ```sh
   sudo pip install -r requirements.txt
   ```
4. Run with `generate.py -h`
